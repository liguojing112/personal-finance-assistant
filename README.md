# 个人财务小助手

一个可定制、本地运行的个人财务分析工具。通过用户自定义分类规则，自动将微信账单支出归类，生成可视化报表，并提供超支提醒。

## 📌 项目背景

市面上的记账软件（如微信记账本）分类规则固定，无法识别个性化消费（如“恒星邦办”属于居住，“抖音电商”属于购物）。手动记账繁琐难坚持。本项目旨在提供一个 **懂你、私密、可扩展** 的财务分析方案。

## ✨ 核心功能

- **账单导入与清洗**：支持微信导出的 Excel 账单，自动清洗、去重、格式化。
- **用户自定义分类**：通过 JSON 文件配置关键词，自由定义支出类别（如“居住”、“餐饮”），大小写不敏感。
- **自动分类统计**：基于规则对每笔支出打标签，输出月度汇总。
- **可视化报表**：生成包含明细、月度汇总、饼图、趋势图、柱状图、超支条形图的 Excel 文件。
- **智能超支提醒**：按月度设置预算，自动检查超支，在报表中高亮显示，并生成超支明细 CSV。
- **一键运行**：整合所有流程，支持批量处理多个账单文件。

## 🛠️ 技术栈

- Python 3.8+
- 数据处理：pandas
- 可视化：matplotlib, openpyxl
- 配置文件：JSON
- 版本控制：Git

## 🚀 快速开始

### 1. 安装依赖

pip install -r requirements.txt

### 2. 配置分类规则

编辑 `config/category_rules.json`，定义你关心的类别及关键词。例如：

{
    "居住": ["水电费", "恒星邦办", "康佳滚筒"],
    "餐饮": ["便利店", "农夫山泉", "肉夹馍", "水果"],
    "购物": ["抖音电商", "拼多多"],
    "社交": ["转账", "红包"],
    "学习": ["打印", "复印"],
    "其他": []
}

> 规则说明：关键词子串匹配（不区分大小写），未匹配的交易自动归为“其他”。

### 3. 配置预算（可选）

编辑 `config/budget.json`，按月份设置预算。未设置的月份将使用 `DEFAULT_BUDGET`（可在 `alert.py` 中修改）。

{
    "2026-01": {
        "总预算": 800,
        "居住": 200,
        "餐饮": 300,
        "购物": 150,
        "社交": 100,
        "学习": 30,
        "其他": 50
    }
}

### 4. 放入账单文件

将微信导出的 Excel 账单放入 `data/raw/` 文件夹（支持多个文件）。

### 5. 一键运行

python src/main.py

如需指定单个文件：

python src/main.py --file "data/raw/微信账单.xlsx"

运行后，在 `output/` 文件夹中会生成：
- `*_report.xlsx`：包含明细、月度汇总、超支统计及多张图表的 Excel 报表。
- `overbudget_details.csv`：所有超支记录明细。

## 📊 输出示例

生成的 Excel 报表包含以下工作表：
- **明细**：每笔交易的详细信息（含类别）。
- **月度汇总**：按月份和类别的支出透视表。
- **超支统计**：列出超支月份、类别、预算、实际及超出金额。
- **图表**：嵌入四张图——支出占比饼图（图例含金额）、月度总支出折线图、月度总支出柱状图、各类别累计超支条形图。

## ⚙️ 高级配置

### 默认预算

当某月份未在 `budget.json` 中定义时，使用 `alert.py` 中的 `DEFAULT_BUDGET`。你可根据自身情况修改该字典。

### 自定义图表样式

图表使用 `matplotlib` 绘制，可在 `report.py` 中调整颜色、尺寸等参数。
